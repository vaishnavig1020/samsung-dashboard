# Samsung Sales Dashboard (Power BI)

## Project Overview
This project presents an interactive **Samsung Sales Dashboard** built using Power BI to analyze sales performance, product trends, and regional insights.

## Objectives
- Analyze overall sales performance
- Identify top-performing products
- Track regional sales distribution
- Monitor key business KPIs

## Tools & Technologies
- Power BI
- SQL
- Excel
- Data Visualization

## Key Insights
- Total Revenue generated from Samsung product sales
- Top-selling product categories
- Regional sales comparison
- Monthly sales trends

- `samsung.pbix` – Power BI dashboard file
